# Как да настроите API ключ за реално AI генериране

## Стъпка 1: Вземете Anthropic API ключ

1. Отидете на https://console.anthropic.com/
2. Създайте акаунт или влезте
3. Отидете на API Keys секцията
4. Създайте нов API ключ
5. Копирайте ключа (започва с `sk-ant-...`)

## Стъпка 2: Добавете ключа в проекта

1. Създайте файл `.env.local` в root директорията на проекта
2. Добавете следния ред:

```
ANTHROPIC_API_KEY=sk-ant-вашият-ключ-тук
```

3. Запазете файла

## Стъпка 3: Рестартирайте сървъра

1. Спрете текущия сървър (Ctrl+C в терминала)
2. Стартирайте отново: `npm run dev`

## Готово!

Сега бутоните ще работят с реално AI генериране от Claude 3.5 Sonnet.

**Важно:** Не споделяйте API ключа си публично! Файлът `.env.local` е в `.gitignore` и няма да се качва в Git.
