======================================================================
   КАК ДА ДОБАВИТЕ ANTHROPIC API КЛЮЧ ЗА РЕАЛНО AI ГЕНЕРИРАНЕ
======================================================================

СТЪПКА 1: ВЗЕМЕТЕ БЕЗПЛАТЕН API КЛЮЧ
-------------------------------------
1. Отворете в браузъра: https://console.anthropic.com/
2. Регистрирайте се или влезте (може с Google акаунт)
3. Кликнете на "API Keys" от менюто
4. Натиснете бутона "Create Key"
5. Копирайте ключа (започва с: sk-ant-api03-...)

СТЪПКА 2: ДОБАВЕТЕ КЛЮЧА В .env ФАЙЛА
-------------------------------------
1. Отворете файла .env в главната папка на проекта:
   C:\Users\deivi\Desktop\Test Project\.env

2. Намерете реда:
   # ANTHROPIC_API_KEY=sk-ant-api03-your-key-here

3. Премахнете # (решетката) и заменете с вашия ключ:
   ANTHROPIC_API_KEY=sk-ant-api03-вашият-ключ-тук

4. Запазете файла (Ctrl+S)

ПРИМЕР:
Преди:  # ANTHROPIC_API_KEY=sk-ant-api03-your-key-here
След:   ANTHROPIC_API_KEY=sk-ant-api03-abc123def456ghi789...

СТЪПКА 3: РЕСТАРТИРАЙТЕ СЪРВЪРА
---------------------------------
1. В терминала натиснете Ctrl+C (за да спрете сървъра)
2. Стартирайте отново с командата:
   npm run dev
3. Отворете http://localhost:3000/dashboard
